# Codex Context

This is the implementation workspace for LearnFlow AI Dashboard. The approved design source has been downloaded from Stitch, and the next major task is to scaffold and build the web app.

## How To Start Future Work

1. Read `AGENTS.md`.
2. Read `docs/product/product-brief.md`.
3. Read `docs/design/responsive-spec.md`.
4. Check `design/stitch/screen-manifest.md`.

## Important Product Decisions Already Made

- The current Stitch design is approved.
- Both mobile and desktop variants exist.
- Mobile design should guide mobile implementation.
- Desktop design should guide desktop implementation.
- The product should be a functional app experience, not a landing page.

## Design Source Policy

Use the Stitch files as reference material. The implementation should be clean app code with reusable components and responsive behavior.
