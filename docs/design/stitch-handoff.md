# Stitch Handoff Notes

## Approved Direction

The approved Stitch design includes both mobile and desktop variants. Mobile is not the only source of truth anymore; desktop variants should guide desktop implementation.

## How To Use The Exports

- Use screenshots for visual matching.
- Use exported HTML to inspect spacing, color, and text content.
- Convert generated structure into maintainable components.
- Do not paste Stitch HTML directly as final app code.

## Design Priorities

- Preserve calm, precise, high-signal UI.
- Keep AI areas distinct but not noisy.
- Keep dashboards dense enough for repeated use.
- Keep journal writing and reading comfortable.
- Keep analytics scannable on desktop.

## High-Risk Screens

- Dashboard: must not become a long desktop scroll.
- Entry Details: needs strong reading/content hierarchy.
- Learning Analytics: must use grid structure well.
- Search Notes: needs useful desktop master-detail behavior.
