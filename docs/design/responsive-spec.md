# Responsive Specification

## Breakpoints

- Mobile: below `768px`.
- Tablet: `768px` to `1023px`.
- Desktop: `1024px` and above.
- Wide desktop: `1280px` and above.

## Global Layout

- Mobile uses a single-column layout and bottom navigation.
- Desktop uses a persistent left sidebar and a constrained main content region.
- Desktop content should use horizontal space with purposeful grids and side panels.
- Avoid scaling mobile screens directly to desktop.

## Screen Rules

### Dashboard

- Mobile: stacked cards in priority order.
- Desktop: summary dashboard grid with metrics, activity, AI insight, recent entries, topics, milestones, and weekly goal.

### New Journal Entry

- Mobile: focused single-column editor.
- Desktop: writing workspace with main editor and right context panel.

### Entry Details

- Mobile: linear reading flow.
- Desktop: reading column plus right panel for AI summary, key concepts, confidence, and related notes.

### AI Coach

- Mobile: chat-first layout.
- Desktop: central chat workspace plus side context panel for topics, suggested prompts, or recent entries.

### Search Notes

- Mobile: search, filters, then result stack.
- Desktop: search/filter header plus result list and preview/synthesis panel.

### Knowledge Quiz

- Mobile: question-first flow.
- Desktop: centered question panel plus progress/session sidebar.

### Learning Analytics

- Mobile: stacked analytics cards.
- Desktop: analytics grid with charts and metrics above the fold.
