# Design System

## Visual Direction

LearnFlow uses a refined productivity-tool aesthetic: clean surfaces, compact cards, subtle borders, precise spacing, and AI-specific violet accents.

## Core Tokens

| Token | Value | Usage |
|---|---|---|
| Background | `#faf8ff` | App background |
| Surface | `#ffffff` | Cards and panels |
| Primary | `#4f46e5` | Primary actions, active nav |
| Primary strong | `#3525cd` | High-emphasis primary states |
| AI accent | `#8b5cf6` | AI coach, AI summaries, magic actions |
| Success | `#10b981` | Progress, streaks, positive deltas |
| Text | `#131b2e` | Primary text |
| Muted text | `#464555` | Secondary text |
| Border | `#e2e8f0` / `#c7c4d8` | Card and control borders |

## Typography

- Primary font: Inter.
- Metadata/status font: JetBrains Mono.
- Use compact, readable dashboard typography.
- Avoid oversized headings inside dense tool surfaces.

## Shape And Elevation

- Small controls: 4px radius.
- Cards and panels: 8px radius.
- Borders should usually be 1px.
- Shadows should be subtle and secondary to border/tonal layering.
