# Routes And Flows

## Proposed Routes

| Route | Feature | Primary Screen |
|---|---|---|
| `/` | Dashboard | Dashboard |
| `/journal/new` | New Journal Entry | New Journal Entry |
| `/journal/:entryId` | Entry Details | Entry Details |
| `/coach` | AI Coach | AI Coach |
| `/search` | Search Notes | Search Notes |
| `/quiz` | Knowledge Quiz | Knowledge Quiz |
| `/analytics` | Learning Analytics | Learning Analytics |

## Navigation

- Mobile: bottom navigation with primary sections.
- Desktop: left sidebar navigation with persistent app identity and section links.
- Primary creation action: New Entry.

## Main User Flows

### Daily Learning Capture

Dashboard -> New Journal Entry -> Save Entry -> Entry Details -> AI Summary.

### AI-Assisted Understanding

Entry Details -> AI Coach -> Ask follow-up -> Save insight or start quiz.

### Retrieval Practice

Dashboard or Quiz -> Knowledge Quiz -> Answer question -> Review feedback -> Continue.

### Search And Synthesis

Search -> Filter notes -> Open result -> Review AI synthesis -> Navigate to Entry Details.

### Progress Review

Dashboard -> Learning Analytics -> Inspect weak areas -> Start quiz or review related notes.
