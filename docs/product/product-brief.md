# Product Brief

## Product

LearnFlow AI Dashboard is a personal learning operating system. It helps users capture learning journals, convert notes into knowledge, receive AI coaching, practice recall with quizzes, and understand learning progress through analytics.

## Target User

- Lifelong learners.
- Professionals learning technical or conceptual topics.
- Users who want a structured system for reflection, spaced recall, and AI-assisted synthesis.

## Core Value

LearnFlow turns daily learning activity into a guided feedback loop:

1. Capture what was learned.
2. Extract concepts and insights.
3. Ask an AI coach for help.
4. Search and reconnect prior notes.
5. Practice with quizzes.
6. Track mastery and learning consistency.

## Core Features

- Dashboard overview.
- New journal entry editor.
- Entry detail and AI summary.
- AI coach chat.
- Search notes and AI synthesis.
- Knowledge quiz.
- Learning analytics.

## Product Tone

The product should feel calm, competent, focused, and high-signal. It should look like a serious productivity tool for learning, not a social feed or marketing site.
