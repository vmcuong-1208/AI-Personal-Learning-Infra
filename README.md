# LearnFlow AI Dashboard

LearnFlow AI Dashboard is a personal learning workspace for journaling, AI coaching, note search, knowledge quizzes, and learning analytics.

This repository is currently prepared as a Codex-friendly implementation workspace. The approved Stitch designs have been downloaded into `design/stitch`, and the product/design/build plans live in `docs`.

## Current Status

- Approved mobile and desktop Stitch screens are available locally.
- Product, design, responsive, architecture, and implementation planning docs are in place.
- `src` and `tests` folders are prepared for the next coding phase.

## Key Folders

- `design/stitch`: Stitch screenshots and exported HTML for all approved screens.
- `docs/product`: Product scope, routes, and user flows.
- `docs/design`: Visual system, responsive behavior, and design handoff notes.
- `docs/plans`: Implementation plan and backlog.
- `docs/agent`: Codex working context and instructions.
- `docs/engineering`: Proposed app architecture.
- `src`: Future application source code.
- `tests`: Future unit and E2E tests.

## Next Step

Pick the frontend stack and scaffold the app. Recommended default: React + TypeScript + Vite or Next.js, depending on whether routing/server features are needed early.

For a fast MVP build, follow `docs/plans/fast-delivery-plan.md`.
