# Stitch Design Source

This folder contains the approved Stitch exports for LearnFlow AI Dashboard.

## Project

- Title: LearnFlow AI Dashboard
- Stitch project ID: `10401796833763224618`
- Source: Google Stitch MCP
- Export method: hosted URLs downloaded with `curl -L`

## Folder Layout

- `screens/mobile/images`: approved mobile screenshots.
- `screens/mobile/html`: generated HTML exports for mobile screens.
- `screens/desktop/images`: approved desktop screenshots.
- `screens/desktop/html`: generated HTML exports for desktop screens.
- `screen-manifest.md`: mapping between product screens, Stitch IDs, and local files.

## Implementation Note

Use screenshots as the visual reference and generated HTML as an inspection aid. The final app should be implemented with maintainable components and responsive layouts, not by pasting generated HTML directly into production.
